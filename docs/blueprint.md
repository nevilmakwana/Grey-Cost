# **App Name**: GreyCost

## Core Features:

- Product Specifications Input: Dedicated input fields for product specifications (Scarf Size, Fabric per Piece, Fabric Price, Printing Price, Shrinkage, etc.).
- Operational Costs Input: Dedicated input fields for operational costs (Stitching Cost, Ironing Cost, Packaging Cost, Delivery Cost).
- Losses & Margins Input: Input fields for losses, overhead and margin (Defective %, Return %, Office Maintenance %, Agent Commission %, Sales Offer %, Profit Margin %).
- Cost Calculation: Automatic calculation of total costs based on input parameters.
- Real-time Selling Price Display: Display the selling price with prominent formatting, updating in real time.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), reminiscent of traditional textile dyes, suggesting quality and craftsmanship.
- Background color: Very light indigo (#F0F2FA), for a clean and modern feel that doesn't distract from the data.
- Accent color: Soft amber (#FFB300), for highlighting key interactive elements, like the Calculate button, and for displaying calculated results like the final price. This evokes the warmth of natural fabrics.
- Font pairing: 'Poppins' (sans-serif) for headers and labels, 'PT Sans' (sans-serif) for input values and the paragraphs of descriptive text.
- Two-column layout: Input fields on the left, results on the right for clear information separation.
- Subtle transition animations to confirm user interactions and enhance the experience.
